# ── LLM / AI crawlers ───────────────────────────────────────────────────────
# Two categories:
#   LIVE-QUERY bots fetch a page when a user asks their assistant a question
#   and usually cite the source. We allow these — they drive discovery.
#   TRAINING bots scrape in bulk to build foundation-model knowledge. We
#   block these; blocking them doesn't hurt current discoverability (live
#   bots still work) but preserves some control over bulk content reuse.
